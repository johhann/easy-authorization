# Easy User, Role, and Permission Management with Spatie and Laravel-authorizer

- clone this repo

`git clone https://github.com/johhann/easy-authorization.git`

- install dependency

`composer install`

- connect your db - setup your .env file

- run migration

`php artisan migrate`

- create your roles

# `Done!`
